ストーリー by hylom 2008年11月21日 18時00分
C-x C-sとC-gですかねぇ、 部門より

iwakuralain 曰く、

アイシェアが、「今年最もお世話になったショートカットキーは？」というお題目の調査結果を発表しています。

調査結果では、Ctrl + CやCtrl + Vといった/.に来る方ならおなじみの組み合わせが上位に来ております。

記事には今年ほとんど使わなかったが便利だと思っているショートカットキーや一度も使ったことがないショートカットキーも載っているので、見てみると新しい発見があるかもしれません。

ちなみに私はShift + Tabの組み合わせを会社の事情でよく使います。Tabキーで行き過ぎた時に便利というのもありますが、いつも会社で使うログイン画面のテキストボックス開始位置がそこで毎日使ってます。

  • 右クリックキー (スコア:1)

    by kawa-t (37052) on 2008年11月21日 21時38分 (#1460041) 日記
    私は右クリックキーにお世話になったのですが、あまり使う人がいないようですね。

    マウスの右ボタンとほぼ同じ機能を持っていて、右クリックメニューを開いて、キーボードから直接メニューを選択するという形で、ショートカットと同じような操作感覚になります。

    Excelの場合ですと、

    右クリックキー、T => カット
    右クリックキー、C => コピー
    右クリックキー、E => コピーしたセル
    右クリックキー、D => 削除
    右クリックキー、S、V => 値のみペースト
    右クリックキー、S、V、M => 値を乗算でペースト
    右クリックキー、S、T => 書式のみペースト
    右クリックキー、T => セルの書式

    などです。マウスの右ボタンと違うのは、選択されているセルに適用される点です。なので、カーソルでセルを選択しているときに、よく使います。もちろん、マウスで選択しているときは、右クリックキーの代わりに、右ボタンをクリックします。

    以前はCtrl系を多用していましたが、キーボードのCtrlキーはえぐれるぐらい使い込んでいて、小指に苦痛を感じることもしばしばでしたので、Ctrlに依存しないキー操作を探していて、行き着きました。

  • Windowsの機能を使って自分で定義 (スコア:1)

    by t-miyoshi (5339) on 2008年11月21日 22時41分 (#1460079)
    Windowsキーを嫌う人は多いですが、Win+Eなんかは便利ですよね。
    これを使いたいけどWindowsキーが無い人は、以下の設定を使うと便利です。

    【アプリケーションのショートカット→プロパティ→ショートカットキー】

    つまりEXPLORER.EXEの機能だけで、自由なショートカット定義ができます。ランチャのインストールも起動も不要。
    ここにCtrl+Alt+Eなどと定義すれば、Win+Eと似たような操作でエクスプローラが起動できます。

    エディタ・コマンドプロンプト・telnet・ワープロ・表計算などなど、自由に定義しちゃいましょう。

    ただし幾つか問題点があります。
    ・多重起動できない
    ・普段使わないユーザやPCでは無効
    ・他のアプリとカブると動かない
    ・Windows9xなんかだとスタートメニュー配下じゃないと動かなかったような…

    この機能を使っている人を見たことが無いのだけど、知られていないもの？それとも不便だから？
