Mac OS X 10.5 Leopardは10月26日発売 111

ストーリー by Acanthopanax 2007年10月16日 22時15分
ポチッと 部門より

YOUsuke 曰く、

Apple StoreにてMac OS X 10.5 Leopardの予約受付が開始された。発売は10月26日午後6時。値段は1万4800円（税込み）。DockやFinderなど基本的なインターフェースが刷新され、また仮想デスクトップであるSpaces、履歴つきバックアップ機能であるTime Machineなど新しい機能も提供される。その他、付属アプリケーションも機能向上がなされる。

はてさて、早速買いか、Leopard搭載の新機種が出るのを待つか。どうしたものか。

プレスリリース「アップル、Mac OS X Leopardを10月26日に出荷」も参考に。

      • Re:Boot Camp正式版 (スコア:3, 参考になる)

        by kozika1983 (32678) on 2007年10月17日 13時23分 (#1235071)
        Mac miniで気になるのが、BootCampでfirewire接続の外付けHDDからWindows XP/Vistaが起動できるかなんですよね。

        これは Windows XP/Vista そのものが、IEEE1394 外付け HDD などリムーバブルデバイスからの起動をサポートしていないので BootCamp 云々以前に無理だと思いますよ (Embedded とかは例外)。
        仮に BIOS とかファームとかで IEEE1394 ドライブを SCSI ドライブや内蔵ドライブのようにエミュレーションしたとしても、Windows の起動中に IEEE1394 デバイスがスキャン・初期化された段階で、ブートドライブとして認識されていたドライブを見失ってしまいます。
        Windows での認識後も IEEE1394 バスに接続されているのにあたかも SCSI デバイスであるかのように振る舞えるような動きが提供されない限りは難しいんじゃないかと。

        # これと同じ理由で、USB につながった機器をスキャンするドライバを組み込んだ MS-DOS 起動ディスクを USB FDD で起動しようとすると、USB FDD 自身が初期化されてルートを見失う、という状況に遭遇したことがあります。
