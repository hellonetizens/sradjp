Mac OS X 10.5で使用できなくなっているSkype、対応バージョンが近く公開される予定 60

ストーリー by headless 2014年08月09日 14時33分
復活 部門より
本家/.「Skype Blocks Customers Using OS-X 10.5.x and Earlier」より

8月5日、事前の予告なくMac OS X Leopardおよび以前のバージョンではSkypeが利用できなくなった。ユーザーはアップデートを指示されるものの、それに従っても問題は解決しない。Skype Communityは苦情であふれている。Skypeの担当者は現在のところOS X Leopard(10.5)で利用可能なビルドがないと説明しているが、今後提供する予定があるのかどうかについては回答していない。

その後、Skypeの担当者がThe Next Webの取材に対し、Mac OS X 10.5用のSkypeは近いうちにダウンロードできるようになると語ったそうだ。Skypeではすべてのプラットフォームで近い将来、古いバージョンのSkypeが使用できなくなることを発表しているが、Windows Phone 7ではSkypeが一切使えなくなるとのことだ(The Next Webの記事、 The Next Webの記事2、 Skype for Windows Phone 7のサポート情報、 本家/.)。
