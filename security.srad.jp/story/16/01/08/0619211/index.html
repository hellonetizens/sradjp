IE 8などのサポートが1月12日で終了、利用時には警告が表示されるように 31

ストーリー by hylom 2016年01月08日 16時10分
しかし微妙にIE9が残るのがなんとも 部門より

Internet Explorer（IE） 8のサポートが来週1月12日で終了する。同時に、Windows 7以降およびWindows Server 2012 R2環境におけるIE 9～10のサポートや、Windows Server 2012環境におけるIE 9のサポートも終了となる（2014年8月のITmediaの記事）。また、Microsoftが1月12に配布する更新プログラム「KB3123303」を適用すると、最新版ではないInternet Explorer（IE）を使っている場合に警告が表示されるようになるという（ITmedia）。

1月12日以降は、「各プラットフォームでサポートされている最新のInternet Explorer」のみがサポート対象となる。
