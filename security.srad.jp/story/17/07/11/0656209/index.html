Windows 7のサポートは2020年1月まで 116

ストーリー by hylom 2017年07月11日 16時30分
UIをコロコロ変えたのが悪い気がする 部門より

今年4月にWindows Vistaのサポートが終了された。次にサポートが終了となるのはWindows 7で、延長サポート期限は2020年1月14日までとなっている。しかし、Windows 7は現在デスクトップOS分野において約5割のシェアを占めており（過去記事）、移行は容易ではない。そのためMicrosoftは移行に向けての取り組みをすでに開始しているという（ITmedia）。

日本マイクロソフトはWindows XPからの移行時の混乱を教訓に、2018年1月より移行についての啓蒙を行い、2018年度中にWindows 10への移行100％を目指すとしている。

6月のデスクトップOSシェア、Windows XPが再び増加

インド政府、Windows 10へのアップグレードを格安で提供するようMicrosoftに要請

Microsoft、Windows XPなど旧OS用の新たなセキュリティ更新プログラムをさらに公開

Windowsのセキュリティ更新プログラム、重要な公共サービスなどで多数使われている限り提供し続ける必要がある？

Windows 10 バージョン1507のサポートが終了

古いプロセッサーを搭載したPCでもWindows 7の更新プログラムがブロックされる問題

4月11日、Windows Vistaがサポート終了を迎える

2年後に迫るWindows 7のサポート終了、少なくない日本企業がまだ移行計画なし

Windows 7、サポート終了に向けた通知表示を4月から開始

Windows 8.1 の延長サポート終了まで 200 日

Windows XP/Vista、5月25日以降iTunes Storeが利用できなくなる

この議論は賞味期限が切れたので、アーカイブ化されています。 新たにコメントを付けることはできません。
