MSが世界の警察に配ったCOFEEって何？ 23

ストーリー by hylom 2008年06月11日 18時35分
Linuxベースだったなら面白かったのに 部門より

Futaro 曰く、

GIZOMODEのこの記事によれば、昨年、世界の警察関係者にMicrosoft社から無償でPCの捜査ツールとして、USBメモリタイプのガジェット「COFEE」が配られたとのこと。


MSのソフトウエアとか、そのソフトウエアで作られたファイルそのものに、バックドアがあって、その情報を使っているのではないか、と友人は推測していましたが。。。

純粋に技術的興味で見てみたいですね。

CNET News.comの「Microsoft serves law enforcement free COFEE | Defense in Depth - computer security, hacking, crime, viruses」という記事によると、COFEEには
  • Incident Response Collection Report
  • First Responder Evidence Disk
  • Windows Forensics Toolchest
  • Forensic Acquisition Utilities
  • Windows Forensic Toolkit
  • Windows Memory Forensics Toolkit
  • The Forensic Toolkit
が含まれているとのことだ。これらはすべて既存のツールで、特に新しいものではない。また、Windows VistaのBit Lockerによる暗号化を解除するツールや、Windowsのバックドアにアクセスするツールが提供されているとの噂もあるが、マイクロソフトの広報はこれを否定している。
