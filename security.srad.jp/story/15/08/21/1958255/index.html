Windows 10の更新プログラムでは重要な変更点しか説明されない？ 83

ストーリー by headless 2015年08月22日 19時08分
闇鍋 部門より
Windows 10では累積的な更新プログラムが随時提供されるようになっているが、Microsoftが重要と考えるもの以外は変更内容について説明されないようだ(The Registerの記事、 BetaNewsの記事、 Neowinの記事)。

MicrosoftはWindows 10の累積的な更新プログラムを8月11日と14日に続けてリリースしている。しかし、11日の更新プログラムに関するKB3081436記事では変更点が明記されているのに対し、14日分のKB3081438記事には「This update includes improvements to enhance the functionality of Windows 10.(この更新プログラムには Windows 10 の機能強化が含まれます。)」とのみ記載されており、具体的な変更内容は説明されていない。これについてThe RegisterがMicrosoftに問い合わせたところ、重要性に応じてほとんどの更新プログラムでは対応するKB記事を公開するといった回答が得られたという。また、更新プログラムにより新機能が提供される場合、新機能の紹介も別途行うとのこと。

Windows 10の標準機能ではインストールする更新プログラムを選択できないが、更新プログラムを指定して非表示化するツールをMicrosoftが公開している。最近のWindows Updateでは繰り返し問題が発生しており、事前に更新の内容を確認したいと考える人も多いだろう。先日トラブルが発生したことが報じられたKB3081424でも、変更内容についての説明は記載されていない。皆さんはどう思われるだろうか。
