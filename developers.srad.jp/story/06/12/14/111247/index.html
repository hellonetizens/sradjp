Java SE 6 リリース 53

ストーリー by Acanthopanax 2006年12月14日 20時15分
改良・向上 部門より

vn 曰く、

日本時間の12日頃、Sun Microsystems から Java SE 6 がリリースされた。主な特徴は以下のとおり。

  1. セキュリティ機能の拡張
  2. いくつかの Web Service 機能の同梱
  3. スクリプト言語サポート
  4. 管理診断機能の向上
  5. 開発生産性の向上
  6. ユーザエクスペリエンスの改良

          気づくはずです。

          のための言語／ライブラリ設計です。Javaは。

          だって「保守」とは詰まる所、ソースの変更です。「書き換え」です。
