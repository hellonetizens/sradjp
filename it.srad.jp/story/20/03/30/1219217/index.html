新型コロナウイルスの解析に協力する分散コンピューティングプロジェクト「Folding@home」、総演算能力1EFLOPS超え 24

ストーリー by hylom 2020年03月30日 12時43分
世界の力 部門より

分散コンピューティングプロジェクト「Folding@home」では新型コロナウイルスの解析に協力することがすでに発表されているが（過去記事）、同プロジェクトに提供されている演算能力が1EFLOPSを超えたことが発表されている（PC Watch、Folding@homeの公式Twitter）。

これは、単純計算でIBMのスーパーコンピュータ「Summmit」の10倍の演算能力に相当するという。

          FAQに定義が書いてありますね。 [foldingathome.org]

          x86 FLOPS are an estimate of how many FLOPS a given calculation would take on an x86 class CPU.

          要するに、x86換算のFLOPSだ、と。
          じゃあ、GPUとかのFLOPSはx86とは異なるのか？ってなるけど

          We refer to the FLOP count on a given hardware as the native FLOP count. For example, an exponential on a GPU is one native GPU FLOP but many native x86 FLOPS.

          意訳：FLOPはそのハードにとってのFLOPである。例えばGPUはexp演算を1FLOPで計算するが、x86は複数FLOP必要だ。
          と書いてあります。
          つまり、GPUでの1FLOPはx86に換算すると数～数十FLOPになる、と。
          この換算係数は演算毎に異なっているらしいんですが（FAQの下の方に係数一覧がある）、換算めんどくさそう。
