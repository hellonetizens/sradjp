Windows Live Messenger、3 月 15 日に廃止 13

ストーリー by reo 2013年01月11日 10時20分
※ただし中国を除く 部門より

ある Anonymous Coward 曰く、

Windows Live Messenger が 2013 年 3 月 15 日に廃止されることがメールにてユーザに通知されている (The Next Web の記事、本家 /. 記事より) 。

Microsoft は 2011 年 5 月にSkypeを 85 億ドルで買収しており、買収以後 Skype と Windows Live Messenger の統合を進めて来た。全世界に 1 億人以上いるという Windows Live Messenger のユーザには Windows Live Messenger の廃止および Skype への移行を通知するメールが配信されているとのこと。廃止以降は Windows Live Messenger にログイン出来なくなるが、同じ ID を使用して Skype にログインでき、連絡先情報も移行されるとのこと。

ただし例外があり、中国では Skype が同国の通信企業である TOM によって運営されているため Windows Live Messenger の提供が継続されるとのことだ。
