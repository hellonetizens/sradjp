メイリオフォントを XP で正式に使えるようになった 34

ストーリー by tach 2007年12月26日 23時13分
VistaのIE7はなぜデフォルトフォントがメイリオじゃないんだろう？ 部門より

alp の日記をアレたまで見て 知ったのだが、Windows Vista の標準フォントであるメイリオフォントが、Windows XP でも正式に利用できるようになったそうだ。 というのは、無料でダウンロードできる Visual Studio 2008 Express Edition 日本語版に メイリオフォントが同梱されているため。 IT Pro の記事や 上記日記にあるとおり、Visual Basic や Visual Web Developer では入らないようで、メイリオが 必要な場合は、Visual C# を入れるといいようだ。

私も実際に試してみたが、最初にインストールしたときはメイリオが入らなかった。 これは、インストール時にオプションを外してしまったから。 「Microsoft Silverlight Runtime」のオプションを付けると ちゃんとメイリオが入ってくれたので、メイリオを使いたい場合は このオプションを付けたままにする必要があるようだ。ともあれ、Firefox の フォントをメイリオにしてみた。 とはいえ、普段は Linux を使っているので、Windows を 使う機会はほとんどないのだが…。

      メイリオは大きなinternal leadingを持っているので従来の日本語フォントとの互換性が低いという大きなデメリットもあります。

      • Word 2003 または Word 2007 でメイリオ フォントを使用した場合に行間が大きく空く [microsoft.com]
      • Bug 5976 – メイリオに対して<abbr>で引かれる点線が下過ぎる [mozilla.gr.jp]

      また、serifやmonospaceに相当するメイリオに似た日本語フォントが用意されていないので、sans-serifのみがメイリオで表示されると非常に変な表示になります。
