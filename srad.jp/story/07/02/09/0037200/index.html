Lenovo、「イノベーション・フォーラム2007」開催 82

ストーリー by yoosee 2007年02月09日 10時56分
遺伝子は受け継がれたのか 部門より

tsukasa_souya 曰く、

Lenovoの項目がなかったので、トピックをIBMにさせていただいたが、 去る2月6日、恵比寿ガーデンホールにおいてLenovoによる「イノベーション・フォーラム2007」が開催された。 レノボジャパン代表取締役社長の天野総太郎氏による挨拶や、レノボグループCEOのウィリアム・アメリオ氏、 ThinkPad開発の中心人物でもあるレノボ・ジャパン副社長内藤在正氏による講演等が行われ、Lenovoの今後についての講演が行われた。

今回の講演では、IBM時代からThinkPadの開発を行っていた大和研が今後もLenovoブランドPCを開発していく事などを繰り返し強調していた。 私の周りでは、如何にThinkPadとはいえ中国企業　(実際は"戦略的統合”との事だが)　になってしまったらそのクオリティを維持出来る物ではないのではないかと懸念する声を良く聞いたが、Lenovoとしても統合による信頼低下がウィークポイントになると理解しているようだ。

今のところLenovoになってからの Thinkpad や Lenovo3000 シリーズにクオリティ低下という話はあまり耳にしていないが、LenovoJapan設立からもうじき2年。 Lenovoについて今一度思考を巡らせてみるのも良いのではないだろうか？

        # Ctrl キーって A の左でしょう?

        ThinkPad は基本的に Windows を搭載して販売されている製品ですし、Windows では Windows キーの有無は地味に利便性を低下させます。Windows キーが無い X30 でも Windows 上では無変換→ Windows キー化して使ってました。

        Windows キーとの組み合わせによるショートカットは地味に便利ですので、慣れるとこれなしで Windows は使えませんし、*BSD などでも meta キーとして利用することは可能です。さりげなく OneNote とかが Win-N とか Win-Shift-N とかにショートカットキーを割り当てていたりするので、この辺りに慣れると Windows キーなしとかありえなくなります。まぁ、もっと使うのは結局 Win-E/L なのですが。

        ThinkPad 愛好者でトラックポイントは不要派の方が圧倒的に少ないと思います。むしろタッチパッド採用モデルなど使いたくも無い、ですね。X シリーズにタッチパッドなど載せられたら重量、面積の面で非常に不利です。

        また、良く採用されている静電式のタッチパッドなどはタイピング中に不意にマウスカーソルが移動したりクリックされたりすることがありますし (X-Mouse 有効にしている人には致命的にうざい)、体質的に静電式タッチパッドは利用できない人もいます。この点を考慮しても、トラックポイントの方が堅実です。

        指紋認証デバイスが中央下部分にあるのが X60/X60s のいいところだと思います。タッチパッド搭載モデルではこの場所に配置できないためホームポジションから指を離す必要がありますが、この位置にあると右手親指や左手親指の「側面」を記録させることで、ホームポジションから指を伸ばすだけで認証できるという、非常に快適な状況を作り出すことができます。この場合、ちょっと認識率悪いですけどね。(w

        指紋認証デバイスは *BSD などでも pam モジュールが配布されているので login などで利用可能です。

        Intel GMA950 って一応 PCIe 接続ですし VRAM も 224MB 程度の割り当てが行われます。下手なビデオチップなど載せられて消費電力や価格を上げられるくらいなら Intel GMA950 で十分だと思いますが、どうでしょうか? Vista でスコア 3.0 とか出ますよ。

        携帯性とパフォーマンスの両立が X シリーズのテーマになっているので、高機能グラフィクスなどでバッテリ持続時間が低下する位なら別のモデルに任せたらいいだけとしか思えません。
