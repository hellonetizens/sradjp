韓国で「Windows と互換性を持つ」という OS「Tmax Window」発表間近 61

ストーリー by reo 2009年06月25日 11時00分
笹の葉ラプソディ 部門より

hylom 曰く、

韓国のソフトウェアメーカーである TmaxSoft が、Windows との互換性を持つという OS「Tmax Window」をリリースするそうだ (CNET Asia Blogs、製品のティザーサイトより) 。この製品は 7 月 7 日に正式発表されるとのこと。

詳細はまだ不明だが、TmaxSoft 社の Web サイトにある「OS」項目には Timos という製品が紹介されており、これは POSIX API および Windows API との互換性があり、組み込みからデスクトップ、エンタープライズ領域までをカバーするものとされている。

CNET Asia Blogs によると、Tmax Window は Linux などの既存の OS をベースにしたものではない、まったく新しい OS とのことで、Tmax Office というオフィスソフトや Tmax Scoutor という Web ブラウザも同時に発表されるそうだ。

韓国では ActiveX が採用されている Web サイトが非常に多いが、Linux では ActiveX を動かすことができない。そこで Tmax Window は「ActiveX が動作する」ことを武器に、小型インターネットデバイスなどへの採用を狙っているそうだ。

なお、TmaxSoft の日本法人 Web サイトでは Tmax Window についての情報はまったく掲載されていない。
