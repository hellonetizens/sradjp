Emacs 用 Vim エミュレータ「Evil 1.0」リリース 36

ストーリー by reo 2013年02月20日 11時20分
don't-be-evil 部門より

ymitsu 曰く、

Emacs のメジャーモードで Vim をエミュレートする「Evil」 (Extensible VI Layer) 初の stable release である stable-1.0 のブランチが作られた (Evil 公式サイト、本家 /. 記事より) 。

Evil は Vim の多くの機能を実装しているほか、Vim のプラグインをもサポートしており、Emacs の Vim 互換モードである「viper-mode」よりもずっと Vim らしい Vim 体験を味わえるようだ。Emacs に不満を持ちながら利用しているユーザーは、一度 Evil を導入してみるとシンプルな幸せを味わえるかもしれない。

ちなみに Emacs ユーザーではない人はコンソールで「vi」と入力した方が手っ取り早い。

よかったよかった。僕も Evil でチルダのコピペミスが減ってモテモテになるかもしれない。
