佐賀の全県立高校、1年生にWindows 8タブレットを5万円自費負担で強制購入させる方針 184

ストーリー by hylom 2013年09月06日 17時15分
Windowsだと考えると 部門より
hauncon 曰く、

佐賀新聞の記事によれば、来年度から佐賀県の全県立高校の1年生を対象に、Windows 8搭載のタブレット端末を自己負担額一律5万円で購入させることが決定されているようだ。

ハードだけでなくデジタル教材も含まれ、実際の価格は5万円を下回ることはないということで、超過分は補助されるようである。記事では経済的困窮家庭や定時制の生徒の負担軽減を問題視しているようであるが、そもそも普及度観点からWindows 8タブレットを記事にあるように標準教材として指定するのは無理があるんじゃないだろうかとも思うのだが、如何なものだろうか？

              • Re:誰が儲かるの (スコア:3)

                by Technical Type (3408) on 2013年09月07日 18時52分 (#2455924)
                貧困の連鎖の防止 [saga-s.co.jp]というのがありますが、生活保護を受けている家庭で育った子どもは、義務教育ではないという理由で高校の学費や教材費、通学費用な支給されません。高校が無償化されなければ、公立高校にさえ通えない境遇の子どもも多かったでしょう。今では、中学を出たらすぐに働けと行政から指導される事は減ったようですが、今も、大学に進学すると生活保護を打ち切られる有様です。中流以上の家庭に生まれるか、貧困家庭に生まれるかは子どもは選べませんが、教育格差はあります。

                低価格なタブレットか「知人が勉強用にとくれたお下がりノートPC」のどちらかさえあれば、これを武器にどんどん学習でき、進学したり手に職を着けて貧困の連鎖から脱出できるようにする事も、理屈の上では可能なんですけどね。一番重要なのは充実した教育コンテンツの蓄積であって、ハードやOSではない。動画が再生でき、EPUB3もPDFも閲覧でき、ブラウザも使えれば十分です。極論を言えば「画像や音や文字を表示したり少しの入力をさせるための端末」なのだから。これと、生徒が選びきれないほどたくさんの有用な教育コンテンツの資産が蓄積されるようにすればいい。「教育システム」として考えた場合、OSやハードの比重は低い。

                国の方針にしても、総務省はフューチャースクール推進事業 [soumu.go.jp]、文部科学省は学びのイノベーション [ict-enews.net]としてますね。名前すら統一できないのかと思うと、両方にお伺いを立てなければいけなくなる下々・現場の困惑顔が目に浮かびます。「先進的ICT利活用教育推進事業」に係る学習者用端末操作体験会を開催しました [saga.lg.jp]を見ると、佐賀県は頑張っているようですね。遠隔授業で災害時や病気の際も学習を進めることができる、デジタル教科書がかなり作りこまれているといった感想が出ているようです。

                ただ、一部の先進的な取り組みに手を挙げる実験校、すなわちITが使える先生が多く新しい取り組みに意欲的という、平均とは大きく異なるレベルの高い学校で、行政のバックアップ付きで、肝いりで、何が何でも「総務＆文部科学省が推進のフューチャースクール＆学びのイノベーションは効果絶大です！」という Ready made answer を出すための学校では上手く行っても、県全部に拡大したら、コケて埃を被っていますとか、ゲーム端末＆エッチ画像鑑賞機になってました、という事は「ありがち」です。

                逆に「デジタル教科書がかなり作りこまれている」といった感想が出るようでは駄目でしょう。つまりこれはデジタル教科書を作り込むために膨大な工数すなわち費用を掛けていて、それであれば見学したシロウトは感銘を受けるというのではぜんぜん駄目。そんな教材は数を揃える事ができず長続きしない。もし他県の別システムでは動かない教材であれば価値は低い。佐賀で作成された有益なコンテンツは沖縄でも大阪でも東京でも北海道でも、そのまま利用できなければ無駄である。導入するデジタル教材、最終価格は入札で決定 [saga-s.co.jp]というのは不安が残る。

                行政として最優先すべきは、学習教材のフォーマットの統一、でしょう。動画は MPEG4 で解像度はいくつ、電子書籍は EPUB3、PDF は解像度いくつ、インタラクテブコンテンツは HTML5 で作成すべし、など。後は文部科学省あるいは省が指定する団体が管理する学習教材配信用の一次サーバー、学校ごとに運用する(クラウドも可)校内配信用の二次サーバーを立てれば、生徒は無償で学校から学習コンテンツをダウンロードし、学習できるようにする。テキストも EPUB3 または PDF で無償配信、とすればよい。(5万円払ってさらに学習教材ごとに金を払えというのではむごい) 端末は動画・EPUB3がスムーズに再生できれば任意だが H.265 対応を奨励、など。先生で独自コンテンツが作れる仕組みの構築も必要かと。タブレットのカメラで動画を撮って編集できる簡易ソフトとか、ワープロ感覚で簡単に EPUB3 と PDF と HTML5 の3通りで出力できるソフトを開発し無償で提供とか。(高額で難易度の高いソフトを揃えなければいけないとなると、ベンダに費用を払う以外に学習コンテンツを作る方法が無いので独自コンテンツが増えない) もちろん、コンテンツ作成は先生方に押し付ければよい、という発想ではなくて。

                佐賀県のは、高性能タブレット＋限られた数のデジタル教材(独自に作りこんだ費用のかかるもの)となる公算が高いですね。教育システムを生かすも殺すも、教育コンテンツの充実次第、という発想が見えないから。まぁ、コケたとしても Word+Excel 練習機としては使える、と割り切れば Windows8.1 もアリかと。でもやはりキーボードは必須ですよね。キーが無いと入力が遅くて授業のノートも取れないし、メールで宿題提出、程度はできないとIT活用とは言えないでしょう。

                マスコミ向けには意味不明の選定理由を言っているようだが、要するにタブレット導入で教育を変えよう! 教育夏まつり2013・二本松 [impress.co.jp]にあるように、他県での導入事例や、NEC とマイクロソフトの取り組み(例: ～教育機関向け～ タブレット端末のススメ [microsoft.com]やWindows in the Classroom [microsoft.com])があれば大規模導入も磐石、Windows のノウハウでサポートができるので地元IT企業でも一部は対応可能、だから大コケする恐れは無い、などの腹づもりで Windows に決めたかな。独自に OS やハードを選定しするノウハウなどないから、MS と NEC が何とかしてくれるのに期待、という事か。(あるいはイベント [kyouikusaikou.jp]に参加して、これの真似をする事に決めた) どうせなら、iPhone/iPadの導入計画が立てづらい [nikkeibp.co.jp]によれば「アップルは、iPhone/iPadに関するサポートライフサイクルを開示していない」などの理由により、大規模導入した歳に3年間安心して使い続ける事ができないという問題がある、などもマスコミに言えば良かったのに。

                でもどうせ被災地で行われたイベントから学ぶなら、防災の備えとしてITを活用して貰いたい。官邸のテレビ会議未接続　保安院、安全委と“断絶”　訓練だけで利用 [47news.jp]という、大失態という表現では済まされない失態も起きたようだが、TV会議(あるいはビデオチャット) は防災時の安否・状況把握など、減災と早期復旧に威力を発揮する。まぁ、停電でルーターなどネット周りの電源が落ちればアウトだが…自家発付きというのも気合が要るが。比較で分かったGoogle「ハングアウト」のSkype、FaceTimeを超える機能 [itmedia.co.jp]も参考になりますね。たとえば Google+ ハングアウトを使えば、他国の高校生とタブレットひとつで交流、というのも可能なようだ。Skype と違い、多地点会議も無償で可能。防災担当の教職員による定例訓練はもちろんだが、「普段使い慣れていない物は非常時には絶対に使えない」ので、学校間で定期的な連絡など事務でも使用し、出張費と時間の削減も兼ねれば良い。とはいえ、Google+ ハングアウトは、Windows では動きませんがね。

                もし“普通の公立中学”がiPadで英語授業を始めたら [itmedia.co.jp]や「タブレット導入」自体を目的化してはいないか？ [itmedia.co.jp]なども参考になる事例だ。でも、9月17日のNHKあなたもハーバード大へ ～広がる無料オンライン講座～ [nhk.or.jp]で放送されるであろう内容は、私がイメージしている事と一番近い。

                無料で受けられるオンライン講座が世界中で急拡大し、教育に革命的な変化をもたらしている。ハーバード大やマサチューセッツ工科大などの名門大学が、こぞって授業を無料配信し修了証も発行。 (中略) さらに途上国では、オンライン講座で貧困層の子供たちの学力アップをはかる取り組みが始まった。一方で、講座で優秀な成績をおさめた人の獲得に大学や企業が乗り出し、(後略)

                佐賀県の教育関係者は、これは必見だろう。佐賀が世界の潮流に逆らったり乗り遅れる理由は何も無い。やはりオンラインによる補助教材ビデオ、大学受験対策となるビデオ、学習困難な生徒への補講ビデオ、資格取得に資するビデオ、卒業後の就職で役立つビデオ、社会のマナーやルールを知るのに有用なビデオ、等を山ほど揃えるのが一番。その発想が無いから、やはり佐賀のは「システムを導入する事が目的」化してしまっている。手品のようなプレゼンに騙され、結局はMSを儲けさせる道具に成り下がりましたね。ロクな教育コンテンツ無き教育システムなど意味は無い。特定のOSの特定の”教育ソフト”でしか動かないコンテンツなど無価値。金を払えば(その金は生徒に出させる)メーカーやベンダが何とかしてくれるだろう、教育側はとシステムなんか分らないので全部任せて、やらせればいい、という発想だな。実際は、教育関係者側による、OSやアプリを選ばない標準フォーマット(MPEG, EPUB3)による教育コンテンツの作成にこそ費用と時間を投じるべきなのだが。

                タブレットは「IT教室」のような所にひとクラス分だけ教材として買えば十分ではないか? 経済困窮家庭の生徒用の自宅学習用には、「Windows XP しか動かない、容易に無償で手に入る PC」に Linux でも入れて、動画・EPUB3・PDF ぐらいが表示できるようにしてあげればビデオは USB メモリで持ち帰れば家にネットが無くても学習できる。一番大切なのは、高校入学にかかる費用諸々に加えタブレット代の5万円もポンと出してくれる親の資力ではなく、努力とやる気で決まる、ようでありたい。
