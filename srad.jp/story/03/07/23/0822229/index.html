    • Re:seamonkey ? firebird ? (スコア:4, 参考になる)

      by Ryuzi Kambe (38) on 2003年07月23日 20時22分 (#364397) ホームページ 日記
      mozillaZine のこの記事 [mozillazine.org]によると、

      Note that 1.5a will be a Mozilla Application Suite release as Mozilla Firebird and Mozilla Thunderbird are not yet ready to become the default applications.

      つまり、Mozilla Firebird と Mozilla Thuderbird はまだデフォルトのアプリケーションとするには未完成なため、1.5a は Mozilla Application Suite [hatena.ne.jp] (=Mozilla Seamonkey) のリリースとなるだろう、とあります。

      さらに具体的なスケジュールが、mozillaZine の 1.5 Alphe リリースの記事 [mozillazine.org]（和訳 [dyndns.org]）内に記載されています。

      まとめると

      ・近々 Mozilla Firebird 0.6 に変更を加えた 0.6,1 をリリース
      ・1.5b の前に Mozilla Thunderbird 0.1 をリリース
      ・Mozilla 1.5 final と同時に Mozilla Firebird 0.7 をリリース
      ・Mozilla Firebird / Mozilla Thunderbird を trunk のデフォルトアプリケーションにする作業は、引き続き行われる

      ということのようです。
