Mozilla 1.0 リリース！ 79

ストーリー by koyhoge 2002年06月06日 3時25分
さぁ、勝負はこれからだ 部門より

Ryuzi Kambe 曰く、 "mozillaZineによると、 Mozilla 1.0がリリースされました。とりあえずリリースノートにはこのリリースでのWhat's New は記載されていませんが、RC3以降のビルドがベースになっているのは間違いないでしょう。

なお、日本の Mozilla コミュニティのもじら組では、昨日サーバを移転したばかりで、まだDNSの更新がいきわたって いないとアクセス出来ませんが、一両日中にはアクセス出来るようになると思いますので Don't Panic.個人的には約3年前から応援して来た、ソースコード公開後の"Mozilla"が 遂にリリースされたかと思うと、感慨を感じなくもないですが、むしろこれからが 楽しくなりそうでとてもわくわくしています。"

リリースと同時に大量のタレコミが。みんな本当に待ってたんだね。(笑)

  • 移転中につき mozillaZine 1.0 リリース記事をこちら (スコア:3, 参考になる)

    by Ryuzi Kambe (38) on 2002年06月06日 12時47分 (#103542) ホームページ 日記
    もじら組サーバだけでなく、mozillaZine日本語訳版をおいているサーバも移転中なのですが、こちらの.toドメインのレジストラは
    48時間経たないとDNSの情報を更新してくれないそうなので、mozillaZine の 1.0 のリリース記念の記事の和訳をこちらに
    ポストしてしまいます。

    --

    Mozilla 1.0 がここに存在する。最後の 1.0 ビルドは、オープンソースコミュニティによって四年間の開発作業を表象するものであり、現在ダウンロードすることができる。この Mozilla 1.0 のリリースは重要な宣言をしている: mozilla.org は今プロジェクトが始まったとき(そして Mozilla 1.0 宣言 [mozilla.org](和訳 [mozilla.gr.jp])の中に秘められた時以来)に指定したゴールについに到達したと信じている。

    Mozilla 1.0 は同時に成熟と信頼性も表している。XUL のような Mozilla の数々の API の多くが固定され、Mozilla toolkit 上でビルドを行う開発者のために保証された互換性が提供される。それに加えて、mozilla.org は 1.0 ブランチをメンテナンスし続ける。この1.0ブランチが、その上で作業するかもしれないベンダーや他のディストリビュータの安定して長命な基盤であるために。更に追加される変更(主としてクリティカルな修正)は、1.0ブランチから 1.0.1 そして 1.0.2 などとしてリリースされるだろう。

    Mozilla 1.0 は mozilla.org のウェブサイト [mozilla.org]の FTP サーバ [mozilla.org]からダウンロード可能である。Mozilla 1.0 のリリースノート [mozilla.org]もアップデートされている。

    MozillaZine はすべての開発者、テスター、バグの報告者、ドキュメントの著者、ベンダーたちと、そして Mozila がこのような成功をおさめることを助けてくれたコミュニティのメンバーに感謝する。我々は Mozilla コミュニティに奉仕した年月と、これからやって来る最新の Mozilla のニュースと主張を伴った増え続ける観衆を提供し続けるであろうこの先の年月を光栄に思う。

    すべての記事... [mozillazine.org]
    --

    - Ryuzi Kambe -

      • Re:ワクワクしないが？ (スコア:1)

        by Stealth (5277) on 2002年06月06日 11時04分 (#103428)

        IE6 から HTML の解釈モードが standard mode と quirk mode に分離。これにより、width プロパティの値がようやく CSS Spec. 通りの containing block の幅で扱ってくれるようになりました。

        quirk mode (IE5.5 以前互換) の場合には、本来の contaning block + 左右 padding + 左右 border の値が width プロパティの値として扱われていたため、CSS でのレイアウトで横幅の算出が IE とその他で異なる結果になっていました。

        詳しくは W3C CSS Level2 Spec. [w3.org] の Box model、Visual formatting model などをどうぞ。
