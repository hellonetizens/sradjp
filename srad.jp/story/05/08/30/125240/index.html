WinFS ベータ版、公開 109

ストーリー by GetSet 2005年08月30日 21時05分
紆余曲折 部門より

KAMUI曰く、"本家記事より。 Microsoft が次期 Windows用統合ファイルシステム「WinFS」のβ1を公開した。

先頃公開された Windows Vista β1用で， デベロッパー向けのリリースは来年に予定されていたが，今回予定を大幅に早めて Microsoft Developer Network (MSDN)の登録者向けに提供されたもの。WinFS は Windows Vista (開発コード：Longhorn)に搭載される予定だったが，開発の遅れから 2006年発売予定の Vista には組み込まれず， 2007年のアドオンリリースで提供される予定になっている。

そんな WinFS だが，Windows XP にバックポートされる予定もあったりして……。"

              読み出せない場合は認証せずに復旧コンソールに入れたりしますが。しかし、復旧コンソールでは C:\、C:\WINDOWS 以下程度しか cd で入れないため、データサルベージには利用できません。(という話じゃないのか)

              ちなみに NTFS を chkdsk で復旧する場合には、step 4/5 までやらないと復旧できない場合があります。しかし、復旧コンソールでは step 3 までしかできない。Windows XP が起動してしまえば step 4/5 までできるのですが、それが起動できない。

              このパターンでは knoppix でもお手上げになるかと思われます。この場合には素直に別のパーティション/ドライブに Windows XP を入れて、そこから chkdsk するのが素直で確実になります。

              knoppix 等では ACL 関係等で書き込みは非常に怖いので、その環境を強引に復旧する必要がある場合には利用できないように思われます。
