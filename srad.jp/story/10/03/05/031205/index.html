北朝鮮の独自OS「Red Star」 72

ストーリー by hylom 2010年03月05日 12時00分
やっぱり「Red」なのね 部門より

あるAnonymous Coward 曰く、

北朝鮮の開発している独自OS「Red Star」について、ロシアの衛星放送局が詳細を報じたそうだ（本家記事）。

Red Starは北朝鮮のIT専門家によって2006年から開発されてきたが、その内容や機能は不明であった。しかし北朝鮮の大学に留学しているロシアの学生のブログによってその詳細が明らかになったとのこと。Red StarはLinuxベースで、Windowsのような見た目とインターフェイスだそうで、デスクトップなどはWindowsそっくりだという。文書作成アプリケーションやブラウザなどのアプリケーションは別ディスクに入ってくるとのこと。「My Country」というブラウザもあり、「My Country BBS」という検索エンジン経由でのみインターネットにアクセスできるもののようだ。

なお、せかにゅではロシア人留学生が公開したスクリーンショットも合せてこの件を報じている。

    by SassyOS2 (8523) on 2010年03月05日 8時33分 (#1727869) ホームページ 日記

    OS "Red Star" - линуксоиды всех стран, соединяйтесь! [livejournal.com](「レッド・スター」OS - Linuxoidy すべての国は団結せよ！：2010.02.27)の記事に基づいた英語記事 [nkeconwatch.com]によると、

    Installation of the Red Star is possible straight from the bootable disk, from hard drive, or via the net. The whole process takes 10 to 15 minutes. While the files are copied, the user is shown tips like in a Windows installation, saying that the system “is now faster and simpler”. Unlike Windows, you will not be allowed to select your system language: only Korean is available.
    レッドスターのインストールは、ブータブルディスクやハードドライブから直接、またはネット経由で可能です。インストールプロセスには10〜15分かかります。ファイルがコピーされている間は、システムが『より速く、簡単に』という見出しと Windows インストーラのようなヒントが表示されます。 Windowsとは異なり、あなたのシステム言語は選択できません：朝鮮語のみ利用可能です。

    ユーザーアカウント [nkeconwatch.com]に使える画像は、韓国の人気漫画のキャラクターだそう（Kim というアカウントはミハイルさんが作成したもの）。

    You will need at least a Pentium III 800 Mhz with 256 Mb RAM and astounding 3Gb hard drive space!
    インストールするには、Pentium III 800MHz の CPU で RAM 256MiB、ハードディスクの空き容量は 3GB が必要です。

    Notice the clock on the left – the year is 99th of the Juche Idea, the official North Korean ideology.
    デスクトップの時計表示の左にある、주체99年という年号表示は北朝鮮の公式イデオロギーです（主体暦 [wikipedia.org]表示）

    The OS has its own keyboard layouts for Korean (does not match the Windows version), English, Russian, Chinese and Japanese.
    この OS がもつキーボードレイアウトは朝鮮語（Windows 版とは異なる）、英語、ロシア語、中国語そして日本語です。

    などなど。

    ところであるスクリーンショットによると、搭載されている Naenara web browser で Firefox 公式サイトにアクセスすると認識されて（GUIはFirefox2）、最新の Firefox 韓国語版(i686 Linux)へアップデートするよう促されたようですが、Mozilla で配布しているソースコード [mozilla.org]は

    アメリカ合衆国輸出管理規制、およびその他の合衆国法の対象となっており、いくつかの国への輸出および再輸出ができません（現在は、キューバ、イラン、リビヤ、北朝鮮（朝鮮民主主義人民共和国）、スーダン及びシリア）。また合衆国からの輸入を禁止されている個人や団体への輸出および再輸出もできません

    とあって、オープンソースとはいえ正規にライセンスされたものではありえないっぽいですね。

      Mozilla で配布しているソースコードは

      アメリカ合衆国輸出管理規制、およびその他の合衆国法の対象となっており、いくつかの国への輸出および再輸出ができません（現在は、キューバ、イラン、リビヤ、北朝鮮（朝鮮民主主義人民共和国）、スーダン及びシリア）。また合衆国からの輸入を禁止されている個人や団体への輸出および再輸出もできません

      とあって、オープンソースとはいえ正規にライセンスされたものではありえないっぽいですね。

      確かに輸出制限があるので「正規にライセンス」はされていないかもしれませんが、これは米国の法律上の問題であって、Mozilla財団の意思ではないはずです。実際、該当ページの英語版 [mozilla.org]では輸出制限の項目は削除 [mozilla.org]されています。(削除に至った経緯は不明ですが…)

      Mozillaプロジェクトのソースコードは、MPL、GPL、LGPLのトリプルライセンスで提供されています [mozilla.org]。これらのオープンソース(あるいはフリーソフトウェア)なライセンスには、特定の団体や個人を除外するような条項は盛り込んではいけない事になっています。

      5. 個人やグループに対する差別の禁止

      ライセンスは特定の個人やグループを差別してはなりません。

      理由: 進化の過程から最大の恩恵を引き出すためには、可能な限り多種多様な人々やグループに、平等にオープンソースに貢献する資格が与えられている必要があります。そこで、オープンソースなライセンスによって誰かを進化の過程から締め出すことは禁止されています。

      アメリカ合衆国を含むいくつかの国では、ある種のソフトウェアに輸出制限を課しています。OSD準拠のライセンスは、適用される可能性がある制限についてライセンス受諾者に警告し、また彼らには法に従う義務があることを示唆しても構いません。しかし、ライセンス自身にそのような制限を取り込んではなりません。

      オープンソースの定義 [opensource.jp]より

      ときとして、政府による輸出統制法規と貿易制裁があなたが国際的にプログラムのコピーを配布する自由を束縛することがあります。ソフトウェア開発者にはこれらの制限を除去したり覆したりする力はありませんが、彼らにもでき、そしてすべきことはプログラムの利用に関する条件としてそういった制限を自ら課すのを拒否することです。こうすれば、制限はこういった政府の管轄が及ばないところの人びとやそこでの活動には影響しないでしょう。
