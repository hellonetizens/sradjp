Microsoft、「Windows XP サポート終了カウントダウン」ガジェットをリリース 49

ストーリー by reo 2011年04月26日 10時30分
XP が許されるのは小学生までだよねー 部門より

cheez 曰く、

2014 年 4 月 8 日の Windows XP サポート終了期限までカウントダウンしてくれるガジェットがマイクロソフトからリリースされた (I Programmer の記事、本家 /. 記事より) 。

「Windows XP から解放されるのを心待ちにしている、そんなあなたには 2014 年の Windows XP のサポート期間終了までの日にちをカウントダウンしてくれるお手軽なガジェットをどうぞ」とのことだが、なんとこのガジェットは Windows Vista もしくは Windows 7 でしか動かないという。

アップグレードを促すのがその目的と思われるが、ジョークなのか本気なのか一瞬考えてしまう。
