11月のデスクトップOSシェア、Windows 8／8.1の合計が初めてWindows XPを上回る 84

ストーリー by headless 2014年12月06日 11時34分
逆転 部門より
Net Applicationsの11月分データによると、Windows 8とWindows 8.1のシェア合計が前月から1.85ポイント増の18.65%となり、初めてWindows XPを上回った(Desktop Top Operating System Versions Share Trend、 Desktop Top Operating System Share Trend、 Desktop Operating System Market Share)。

Windows 8.1は単体でも1.18ポイント増の12.1%となり、3.61ポイント減で13.57%のWindows XPに1.47ポイント差となった。Windows 8のシェアは0.67ポイント増の6.55%。Windows 7も3.36ポイント増やして56.41%となっている。Mac OS X 10.9のシェアは0.87ポイント減の2.79%となり、1.48ポイント増で2.66%のMac OS X 10.10が0.13ポイント差まで迫っている。こちらは近いうちに順位が入れ替わることになるだろう。Mac OS全体のシェアは0.22ポイント増の7.27%となっている。このほか、Windows Vistaは0.17ポイント減の2.65%、Linuxは0.16ポイント減の1.25%。Windows全体のシェアは0.05ポイント減少して91.48%となった。

StatCounterのデータでもWindows 8.1は1.82ポイント増の12.12%となり、1.38ポイント減で11.84%のWindows XPを初めて上回っている。ただし、Windows 8は1.14ポイント減で5.42%となっており、Windows 8/8.1合計のシェアは0.68ポイント増の17.54%にとどまっている。その他のOSの変動は少なく、Windows 7が0.82ポイント増の55.75%となったほか、OS Xが0.15ポイント減の8.98%、Windows Vistaが0.02ポイント増の2.99%、Linuxが0.08ポイント減の1.33%。一方、日本ではWindows 8.1が1.13ポイント増の15.49%となったが、Windows 8は1.07ポイント減の2.73%となり、Windows 8/8.1合計のシェアは0.06ポイント増の18.22%にとどまる。その他のOSはWindows 7とWindows Vistaが減少傾向、OS Xが増加傾向にあるが、いずれも増減は1ポイント以下となっており、大きな変化はみられない。
