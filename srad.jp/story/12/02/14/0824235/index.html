Windows 8の新機能の一部は「Linuxの真似」？ 112

ストーリー by hylom 2012年02月14日 17時30分
さすがに一部は言い過ぎのような…・・ 部門より
eggy 曰く、

Windows 8開発の進捗状況を細かく追っているというSandro Villinger氏がIT World記事で、Windows 8の新機能にはLinuxの模倣が見受けられると述べている（本家/.、IT World記事）。

真似したとされる新機能は以下の8個。

  • ファイルコピーダイアログ
  • ISOマウント
  • USBメモリでWindows環境を持ち歩ける「Windows to Go」
  • Windows Phoneに似たインターフェースの「Metro UI」
  • ソーシャルメディアとの統合（ただこれに関しては改善点ではなく、世のトレンドに遅れをとっていたMicrosoftがようやくトレンドに追いついたとしている）
  • USB 3.0のネイティブサポート
  • クラウド「SkyDrive」との統合
  • 復元を強化した新ファイルシステム「Resilient File System」（ReFS）

Liniuxの機能を模倣したとするのか、それとも改良したとするのか意見が分かれるところであろう。
