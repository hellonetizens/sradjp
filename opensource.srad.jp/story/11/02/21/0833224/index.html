P2P匿名掲示板「新月」、いくつかの新機能が実装された新バージョンのサーバントを公開 33

ストーリー by hylom 2011年02月21日 18時17分
利用者が増えれば面白そう 部門より

あるAnonymous Coward 曰く、

P2Pネットワークを用いた匿名掲示板を開発している新月プロジェクトが2011年2月20日、サーバント実装である「朔」安定版の新バージョン3.10.0をリリースした。

このバージョンからは掲示板の配色・デザインが変更され、2ちゃんねる風のものから青を基調としたものに変わる。また2ちゃんねるなどのスレッドフロート型掲示板に存在する投稿があったスレッドをスレッド一覧の上部へ並べない機能（sage）や、過去ログを取得しないでスレッドを開く機能、あとで過去ログを取得する機能が実装された。

最近はノードやユーザー、開発者それぞれ増え始めており、今後も注目されるべきプロジェクトだろう。

      • Re:P2Pって言うのか？ (スコア:4, 興味深い)

        by ruto (17678) on 2011年02月22日 2時04分 (#1906258) 日記
        匿名性確保に関してですが、更新のプロセスは以下のような感じになります。
        (以前調べたときのものなので、今は違うかも。更新情報を見るかぎり大幅には変わってないはず)

        自分のノードを更新する。
        更新したという情報を隣接ノードに送信する。
        隣接ノードは、自分がそのスレッドを購読している場合以下のようにする {
            元ノードに更新内容問い合わせる。
            返ってきた結果で自分を更新する。
            更新元ノードを自分にして更新情報を別の隣接ノードに送る。
        } 購読していない場合 {
            更新元ノードを変えずに更新情報を別の隣接ノードに送る。
        }

        また、最初のノードは更新日時を本当の更新日時からランダムに過去にずらします。
        更新情報を受け取ったノードは、そこに書かれている更新元ノードが最初の更新をしたノードなのか単にリレーしたノードなのか区別が付きません。

        ※以下個人的見解
        ただ、ISPレベルでネットワークを監視されたり、そうでなくても大量の監視用ノードを作られた場合、匿名性はかなり下がると思います。
